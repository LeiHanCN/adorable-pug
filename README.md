# adorable-pug
优雅的静态页面模板

## 使用之前
- 项目使用了 [node.js](https://nodejs.org/zh-cn/) 服务，请先提前安装 [node.js](https://nodejs.org/zh-cn/)

## 开始使用
```bash 
# clone 模板
git clone https://github.com/LeiHanCN/adorable-pug.git <your-project-name>

# 进入到脚本路径下，初始化项目
cd <your-project-name>/scripts
chmod 777 init.sh
./init.sh

# 回到项目目录下
cd ..

# 安装依赖
yarn 
# 开发
yarn dev
# 打包
yarn build
```

## 依赖的技术栈
- babel
- stylus
- pug
